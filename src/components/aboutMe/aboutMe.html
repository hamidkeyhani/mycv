Note: when you set an ARGB color to ColorWheel it's transformed to the closest HSV color with the value(brightness) component of it set to 1 so the color can be correctly displayed on ColorWheel since it's only 2 dimensional. Therefore the color that is displayed by ColorWheel may differ from the original color which you have set. To change the brightness or the alpha component of a color picked from ColorWheel you can use GradientSeekBar view.
